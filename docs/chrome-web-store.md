# Chrome Web Store Listing

## Name
AI Fake Or Real

## Short description (132 chars max)
Know if what you're reading was made by AI. Works on Reddit, Instagram, YouTube, Facebook, and more.

## Detailed description
AI Fake Or Real tells you whether the content you're viewing was generated by AI — instantly, as you browse.

HOW IT WORKS
The extension uses vocabulary analysis, structural pattern detection, and community signals to assess whether text content was written by AI. It checks for telltale AI writing patterns like overuse of words like "delve" and "leverage," uniform sentence structure, and lack of contractions — patterns that human writers rarely produce together.

FEATURES
• Instant verdict: Green checkmark (human), amber question mark (unclear), or red "AI" badge on every page
• In-page indicators: On Reddit, Instagram, YouTube, Facebook, and X, see small AI badges directly on flagged posts as you scroll
• Signal breakdown: See exactly WHY content was flagged — which specific AI patterns triggered
• Works behind login walls: The extension reads the page you're viewing, so it works on content a server can't access
• Comment detection: Spots when other users are calling content AI ("this is AI generated", "AI slop")
• Report incorrect: Flag wrong verdicts so we can improve
• Free public API: Developers can integrate AI detection into their own apps

PRIVACY
• No browsing history collected
• In-page scanning happens entirely in your browser (no data sent to servers)
• Server only contacted when you explicitly click Scan or open the popup
• See our full privacy policy in the extension

SUPPORTED PLATFORMS
Reddit, Instagram, YouTube, Facebook, X (Twitter), TikTok, and any website with text content.

## Permission justifications

| Permission | Justification |
|-----------|---------------|
| activeTab | Read the current page's text content when the user clicks "Scan" |
| storage | Store user preferences (auto-check, overlay toggle, API URL) and cached scores |
| tabs | Update the toolbar badge color/text based on the current page's AI score |
| host_permissions (localhost) | Communicate with the local development backend during testing |
| optional_host_permissions | Allow the user to configure a custom API server URL |
