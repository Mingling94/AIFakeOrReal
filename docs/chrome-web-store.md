## Name
AI Fake Or Real

## Short description (132 chars max)
Detect AI-generated text, images, and video as you browse. Powered by LLM analysis, heuristics, and community votes.

## Detailed description
AI Fake Or Real tells you whether the content you're viewing was generated by AI — instantly, as you browse.

🔍 HOW IT WORKS
The extension runs a three-layer detection pipeline on every page:

1. Local heuristic scan (instant, private, no data sent anywhere) — detects AI vocabulary patterns, structural uniformity, and community comments calling content AI.
2. LLM-powered classification — sends page text to AI models (GPT-4o, Claude, Gemini, and more) for deep analysis. Results are cached so repeated visits don't cost extra.
3. Community votes — see what other users think, and cast your own vote.

These three signals combine into a single verdict: Real, Unclear, or AI Fake.

✨ FEATURES
• Instant badge: Green ✓ (Real), amber ? (Unclear), or red AI (AI Fake) on every tab
• Multimodal detection: Analyzes text, images, and video — not just text
• Smart caching: LLM results cached for 1 week — 10 users viewing the same page only trigger 1 AI call
• AI Exposure stats: See what % of your browsing contains AI content (1 day, 7 days, 30 days, 1 year)
• In-page overlays: On Reddit, Instagram, YouTube, Facebook, TikTok, and X, see small badges directly on flagged posts as you scroll
• Signal breakdown: See exactly WHY content was flagged — which specific AI patterns triggered
• Crowd voting: 👍 Real or 👎 AI Fake — your vote helps improve detection for everyone
• Disagreement alerts: When the scanner and community disagree, you'll see an ⚡ notice
• Dark mode: Automatically matches your system theme
• Keyboard shortcut: Cmd+Shift+A (Mac) or Ctrl+Shift+A (Windows) to open the popup

🔒 PRIVACY
• No browsing history collected — AI exposure stats are stored locally, never sent to any server
• In-page scanning happens entirely in your browser (no data leaves your machine)
• URL hashes (not full URLs) are used for score lookups
• No cookies, no tracking, no ads
• GDPR compliant — delete your data anytime
• Full privacy policy: https://github.com/Mingling94/AIFakeOrReal/blob/main/docs/privacy-policy.md

🌐 SUPPORTED PLATFORMS
Reddit, Instagram, YouTube, Facebook, X (Twitter), TikTok, and any website with text or image content.

## Permission justifications

| Permission | Justification |
|-----------|---------------|
| activeTab | Read the current page's text and images when the user opens the popup |
| storage | Store user preferences, cached scores, and local AI exposure history |
| host_permissions (Railway URL) | Communicate with the AI Fake Or Real backend API for score lookups and analysis |
| optional: tabs | Update the toolbar badge when switching between tabs (not required — badge still works on page load without this) |
| optional: host_permissions (all URLs) | Allow the user to configure a custom API server URL |
