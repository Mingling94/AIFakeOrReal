# Privacy Policy — AI Fake Or Real

**Last updated:** June 2026

## What we collect

- **URL hashes:** When you click "Scan" or open the popup, we send a SHA-256 hash of the page URL to our server to look up or store the AI detection score. We do NOT send the full URL unless you explicitly trigger analysis (which requires fetching the page content).
- **Page content for analysis:** When you click "Scan this page," the extension reads the visible text and comments on the page you're viewing and sends it to our server for AI detection analysis. This content is used only for analysis and is not stored permanently.
- **Votes and reports:** If you vote (thumbs up/down) or report an incorrect verdict, we store the vote/report associated with the URL hash. If you're logged in, it's linked to your account; otherwise it's anonymous.
- **API key usage:** If you use the public API with a key, we count the number of requests for usage tracking.

## What we do NOT collect

- **Browsing history:** We do NOT track which pages you visit. The extension only communicates with our server when you interact with it (open the popup, click Scan, vote, or report).
- **Personal information:** No names, emails, or identifying information is collected unless you create an account (which is optional and only for weighted voting).
- **Cookies or tracking:** We do not use cookies, analytics trackers, or third-party tracking services in the extension.

## In-page scanning

When enabled, the extension scans visible post text on supported social media platforms (Reddit, Instagram, YouTube, Facebook, X, TikTok) using a local, client-side heuristic — this analysis happens entirely in your browser with NO data sent to any server. Only posts that score above the AI-detection threshold are visually flagged. You can disable this in the extension's options.

## Data storage

Detection scores and vote counts are stored on our server associated with URL hashes (not full URLs). Account data (email, hashed password) is stored if you create an account. All data can be deleted on request.

## Third parties

We do not sell, share, or provide your data to any third party.
