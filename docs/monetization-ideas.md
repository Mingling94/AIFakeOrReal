# Monetization Ideas

Current plan: **Free.** Build a useful app, get users, grow.

We'll collect monetization ideas here for when the time comes.

## Ideas

| Idea | Notes |
|------|-------|
| Usage-based API pricing | Free tier generous; charge high-volume API consumers. Metering already built. |
| Pro tier (extension) | More detailed analysis, priority scanning, export reports. Surface only in options page. |
| Enterprise API | Bulk analysis, webhooks, SLA. Separate pricing page. |
| | |

## Principles

- Never gate the core verdict behind a paywall
- Never show monetization in the popup or in-page overlays
- Free tier must be genuinely useful, not a teaser
- Growth first, monetization later
