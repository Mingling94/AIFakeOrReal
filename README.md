# AI Fake Or Real

> Crowdsourced AI content detection for any URL on the internet.

A Chrome extension that combines **community votes** and **AI analysis** to tell you whether the content you're reading was generated by AI.

## Features

- **One-glance badge** вҖ” green (Human), yellow (Mixed), or red (AI) on every tab
- **Crowdsourced voting** вҖ” vote Human / Mixed / AI Generated on any page
- **AI text analysis** вҖ” statistical heuristics detect AI writing patterns (perplexity, burstiness, vocabulary richness, sentence uniformity)
- **Combined scoring** вҖ” AI model score + crowd votes, dynamically weighted by vote count
- **Community stats** вҖ” see how others voted with a visual breakdown
- **Accounts & reputation** вҖ” voter accuracy tracking and reputation system

## Quick Start

### Prerequisites

- Docker & Docker Compose
- Node.js 20+

### Backend

```bash
docker-compose up
```

The API will be available at `http://localhost:8000`. Interactive docs at `http://localhost:8000/docs`.

### Extension

```bash
cd extension
npm install
npm run build
```

Then in Chrome:
1. Go to `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `extension/dist` folder

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Extension | TypeScript, React 18, Webpack, Chrome Manifest V3 |
| Backend | Python 3.12, FastAPI, SQLAlchemy, Alembic |
| Database | PostgreSQL 16, Redis 7 |
| Auth | JWT (python-jose) + bcrypt |
| AI Detection | Statistical NLP heuristics (numpy) |
| CI | GitHub Actions |

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/v1/score?url=` | Get combined score for a URL |
| POST | `/api/v1/vote` | Submit a vote |
| GET | `/api/v1/votes?url=` | Get vote breakdown |
| POST | `/api/v1/analyze?url=` | Trigger AI analysis |
| GET | `/api/v1/analysis?url=` | Get analysis details |
| POST | `/api/v1/scores/batch` | Batch score lookup |
| POST | `/api/v1/auth/register` | Register |
| POST | `/api/v1/auth/login` | Login |
| GET | `/api/v1/auth/me` | Current user info |
